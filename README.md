# 哲风壁纸爬虫

从 [哲风壁纸](https://haowallpaper.com/)（haowallpaper.com）按分类批量下载壁纸的桌面工具，支持图形界面与命令行，可打包为独立 exe。

## 功能

- **分类筛选**：与官网顶部筛选一致，可组合使用
  - 种类：静态 / 动态壁纸
  - 分类：动漫、风景、美女等（启动时自动同步官网）
  - 分辨率：1K～33K
  - 色系：偏蓝、偏绿、黑白等
  - 标签、关键词：如「动漫」「风景」
- **数据源**：电脑壁纸（`/homeView`）、手机壁纸（`/mobileView`）
- **爬取程度**
  - 快速：仅列表缩略图，不访问详情页
  - 标准：详情页预览图（推荐，无需登录）
  - 完整：含动态壁纸 MP4
- **其他**：自定义页码范围、保存目录、请求间隔；跳过已下载文件；运行中可停止

## 项目结构

```
├── main.py           # 打包入口（英文文件名，避免 PyInstaller 识别问题）
├── 爬虫.py           # 爬取核心：分类 API、列表解析、下载逻辑
├── 爬虫_gui.py       # Tkinter 图形界面
├── 哲风壁纸爬虫.spec  # PyInstaller 打包配置
├── build.bat         # 一键打包脚本
└── requirements-build.txt
```

## 环境要求

- Python 3.10+
- Windows（GUI 与打包脚本面向 Windows；核心逻辑可跨平台运行）

## 安装与运行

### 1. 安装依赖

```bash
pip install requests pycryptodome
```

> `pycryptodome` 用于解密官网分类接口，缺少时无法加载分类下拉选项。

### 2. 启动图形界面

任选其一：

```bash
python main.py
python 爬虫_gui.py
python 爬虫.py
```

### 3. 命令行模式（无 GUI）

```bash
python 爬虫.py --cli
```

## 使用说明

1. 启动后等待「已同步官网分类」提示（需联网）。
2. 选择 **电脑 / 手机** 壁纸类型。
3. 在 **分类筛选** 中按需选择种类、分类、分辨率、色系，或填写标签 / 关键词。
4. 设置 **页码范围**（每页约 12 条）与 **爬取程度**。
5. 点击 **开始爬取**；文件默认保存到 `wallpapers` 目录（可在界面修改）。

未选择任何筛选时，程序会提示确认，避免误爬全站大量数据。

## 打包为 exe

```bash
build.bat
```

或手动执行：

```bash
pip install pyinstaller requests pycryptodome
pyinstaller --noconfirm --clean 哲风壁纸爬虫.spec
```

成功后生成：`dist\哲风壁纸爬虫.exe`（单文件、无控制台窗口，约 17 MB）。

exe 可在无 Python 环境的 Windows 上直接运行；首次使用仍需联网以加载分类。
